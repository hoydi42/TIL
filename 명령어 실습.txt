1. 바탕화면에 
practice.txt 파일 만들기
(touch 명령어)

2. 파일 있는지 확인하기
(ls 명령어)

3. CLI 디렉토리 만들고,
CLI로 이동하기
( mkdir 명령어 / mv 명령어)

4. CLI 디렉토리로 경로 이동하여, 
파일이 잘 옮겨 졌는지 확인
(cd 명령어, ls 명령어)

5. practice 파일을 삭제
(rm 명령어)
* tab을 통해 자동완성가능

6. 다시 바탕화면 경로로 이동
(cd 명령어)
- 상대경로 : cd ..
- 절대경로 : cd /c/Users/student/Desktop

7. 더 알아두면 좋은 명령어
pwd : 현재 위치
